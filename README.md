# OpenClaw Avatar Maker

Add OpenClaw logo overlay to avatar images

**Live:** [yuler.github.io/avatar-for-openclaw](https://yuler.github.io/avatar-for-openclaw/)

## Tech Stack

- [Vite](https://vitejs.dev/)
- [Svelte](https://svelte.dev/)
- [Tailwind](https://tailwindcss.com/)

## Development

```bash
pnpm install && pnpm run dev
```
